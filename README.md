# LAB-OOP-4-2026 # 
Repositori Tugas Praktikum Pemrograman Berorientasi Objek 2026

## 1. Fork Repositori ##
Lakukan Fork pada repositori utama ini ke akun GitHub Anda melalui tombol "Fork" di pojok kanan atas halaman.
## 2. Clone Repositori ##
Salin repositori hasil fork ke komputer lokal Anda:
```
https://github.com/Haris2512/LAB-OOP-4-2026.git
```
## 3. Buat Branch Baru ##
Masuk ke direktori proyek dan buat branch baru menggunakan 3 digit terakhir NIM Anda:
```
cd LAB-OOP-4-2026
git branch XXX
git checkout XXX
```
*Catatan: Ganti XXX dengan 3 digit NIM Anda. Pastikan tanda asterisk (*) sudah berpindah ke branch tersebut saat dicek dengan git branch.*
## 4. Struktur Folder Tugas ##
Simpan pekerjaan Anda pada path berikut:
```
app/src/main/java/pbo26/week_x
Contoh: week_1, week_2, dst.
```
Gunakan template file yang telah disediakan.

5. Setelah mengerjakan, stage pekerjaan Anda dengan perintah:
```
git add .
```
6. Setelah pekerjaan selesai, lakukan commit dengan pesan yang sesuai:
Pesan commit di atas hanyalah contoh, silakan sesuaikan dengan tugas yang
dikerjakan.
```
git commit -m"'feat': tugas praktikum-1"
```
7. Lakukan push ke repositori masing-masing:
```
git push origin xxx
``` 
8. Lakukan pull request pada repositori utama. Pastikan base branch dan
compare branch sesuai dengan NIM Anda masing-masing.
